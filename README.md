\# Mini-projet Calculette Web (Git Flow)



\## Présentation

Ce mini-projet a pour objectif de développer une calculette web simple permettant

d’effectuer des opérations arithmétiques de base, tout en appliquant un workflow Git

de type Git Flow.



Le projet a été réalisé dans le cadre d’un partiel sur Git.



\## Fonctionnalités

\- \[x] Addition

\- \[x] Soustraction

\- \[ ] Multiplication



\## Structure du projet

.

├── index.html

├── style.css

├── calculator.js

├── README.md

└── .gitignore



\## Organisation Git

\- main : branche stable

\- develop : branche d’intégration

\- feature/addition : implémentation de l’addition

\- feature/subtraction : implémentation de la soustraction

\- feature/multilplication : implémentation de la multiplication



Un conflit a été volontairement créé dans le fichier README.md puis résolu lors de la

fusion des branches.



\## Lancer le projet

Ouvrir le fichier index.html dans un navigateur web.

Projet réalisé dans le cadre d’un partiel Git / Git Flow.
